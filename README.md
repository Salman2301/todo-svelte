Todo App using Svelte
<br>
This app is not connected to DATABASE
<br>
This app contain motivation text + Clock + todo checklist.

### How to use

1. Clone or download this repo
2. open the terminal in the downloaded dir
3. run `npm install`
4. run `npm run dev`
